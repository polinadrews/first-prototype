Research scenario

This user has previously created an account and is now returning to apply for a visa.

We want to know if they have understood:

  1. how to leave and return to their account or application
  2. what they understand when they go back to their application
  3. whether the current state of the application journey within an account makes sense
    • What doesn't work?
    • What breaks the user's journey?
    • What is missing?
  4. the link of the task list between the account
